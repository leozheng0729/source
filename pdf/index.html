PDF cleanup that feels safe enough for real documents.

FAQ

Questions people ask before installing.

Does this upload my PDFs?

No. PDF operations are handled locally in the browser. The extension is built for documents you do not want to send to random online PDF tools.

What can I do for free?

Free users can handle common single-file PDF tasks such as split, extract, rotate, watermark and compress.

What does Pro unlock?

Pro unlocks batch processing, folder import, no file size limit, no footer mark, batch rename, metadata cleanup, manual redaction and one-click secure ZIP export.

Can it redact scanned PDFs?

Manual redaction works on visible page areas, including scanned pages, because you draw the black boxes yourself. OCR-based automatic detection is not required.
